# FlightScope 3D

Single-file Three.js flight visualizer with a rotatable Earth, starfield, simulated aircraft telemetry, smooth position interpolation, route trails, and responsive HUD controls.

## Files

- `index.html` - Complete app, including HTML, CSS, and JavaScript.

## Local Preview

Open `index.html` in a browser. An internet connection is required for the Three.js CDN imports.
